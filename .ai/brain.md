# OrderKart Project Brain

Welcome to the OrderKart project permanent knowledge base. This file serves as the main entry point to understanding the project's features, domain model, and ongoing developmental trajectory.

## Project Vision
OrderKart is a high-performance offline-first local business order management and worker synchronization app built with Flutter. It is designed to run in environments with low or intermittent internet connectivity, featuring local SQLite storage and P2P hot-spot synchronization between worker devices and the store owner.

## Key Project Themes
1. **Apple-style Glassmorphism**: Premium frosted-glass visual design propagated throughout the application.
2. **Offline-First & Local DB**: Sqflite (SQLite) database mapping with robust fallback synchronization.
3. **Owner ↔ Worker Scoping**: Workers get scoped sub-data provisioned by the Owner to prevent worker-to-worker or unauthorized global access.
4. **P2P Synchronization**: Local Wi-Fi hotspot sync service to transfer data safely.
